# Day 01 - Cloud Concepts

## Características de Cloud Computing
-   Agility;
-   Maintenance;
-   Reliability;
-   Security;
-   Performance;
-   Scalability;
-   Cost and Elasticity.

## Características de Cloud Computing segundo o NIST (The National Institute of Standards and Technoligy)
-   On demand self-service;
-   Broad network access;
-   Resource pooling;
-   Rapid elasticity;
-   Measured service.
